Dataset

We utilized the following Kaggle dataset containing US Covid-19 data from 2020 for our project. We wanted to work with Covid-19 data as this is a topic that is relevant to our day to day lives and that is a current event and thus will have some new data available. We chose this dataset because it provided us multiple insights into the effects of the pandemic on the population whether it be through hospitalizations, persons in the ICU, deaths, etc. Additionally, the data represents an entire year which allowed us to see changes over time from before we knew about the pandemic through its peak to the end of the year.
